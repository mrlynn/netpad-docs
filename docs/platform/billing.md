# Billing & Subscriptions

NetPad supports subscription-based billing via Stripe integration.

## Subscription Plans

### Free Plan

- **Forms**: Limited number
- **Submissions**: Limited per month
- **Basic Features**: Core functionality
- **Community Support**: Community help

### Pro Plan

- **Unlimited Forms**: No form limits
- **Higher Limits**: More submissions
- **Advanced Features**: Workflows, AI
- **Priority Support**: Faster support

### Enterprise Plan

- **Everything in Pro**: All Pro features
- **Custom Limits**: Tailored to needs
- **Dedicated Support**: Personal support
- **Custom Features**: Tailored features

## Managing Subscriptions

### Viewing Subscription

1. **Go to Billing**:
   - Organization settings
   - Click "Billing" tab
   - View current plan

2. **Subscription Details**:
   - Current plan
   - Billing cycle
   - Next billing date
   - Usage statistics

### Upgrading Plan

1. **Select Plan**: Choose new plan
2. **Review Changes**: See what's included
3. **Enter Payment**: Add payment method
4. **Confirm Upgrade**: Complete upgrade

### Downgrading Plan

1. **Select Plan**: Choose lower plan
2. **Review Impact**: See what you'll lose
3. **Confirm Downgrade**: Complete downgrade
4. **Effective Date**: Changes at end of cycle

## Payment Methods

### Adding Payment Method

1. **Go to Billing**: Billing settings
2. **Payment Methods**: Manage methods
3. **Add Card**: Enter card details
4. **Save**: Payment method saved

### Updating Payment Method

1. **Select Method**: Choose to update
2. **Edit Details**: Update information
3. **Save**: Changes saved
4. **Verify**: Confirm update

## Usage Tracking

### Tracked Metrics

- **Forms**: Number of forms
- **Submissions**: Submissions per month
- **Workflows**: Number of workflows
- **Storage**: Data storage used
- **API Calls**: API usage

### Usage Limits

- **Plan Limits**: Based on plan
- **Overage**: Charges for overage
- **Notifications**: Warn when approaching limits
- **Upgrade Prompt**: Suggest upgrade when needed

## Invoices

### Viewing Invoices

1. **Billing History**: View past invoices
2. **Download**: Download PDF invoices
3. **Email**: Receive invoices via email
4. **Details**: See invoice line items

### Payment History

- **Past Payments**: All payment history
- **Payment Methods**: Methods used
- **Receipts**: Download receipts
- **Tax Information**: Tax details

## Cancellation

### Canceling Subscription

1. **Go to Billing**: Billing settings
2. **Cancel Subscription**: Click cancel
3. **Confirm**: Confirm cancellation
4. **Effective Date**: When cancellation takes effect

### After Cancellation

- **Access Until**: Access until period ends
- **Data Export**: Export your data
- **Downgrade**: Reverts to free plan
- **Reactivation**: Can reactivate anytime

## Best Practices

1. **Monitor Usage**: Track usage regularly
2. **Plan Ahead**: Upgrade before hitting limits
3. **Review Bills**: Check invoices regularly
4. **Update Payment**: Keep payment method current
