# Data Explorer Overview

The Data Explorer provides a visual interface for browsing, managing, and manipulating MongoDB collections. No need for MongoDB Compass or command-line toolsâ€”manage your data directly in NetPad.

## What is Data Explorer?

Data Explorer lets you:
- **Browse Collections**: Navigate MongoDB collections visually
- **View Documents**: See documents in multiple formats
- **Edit Data**: Modify documents with schema-aware editor
- **Search & Filter**: Find documents quickly
- **Import/Export**: Bulk data operations
- **Manage Collections**: Create, rename, delete collections

## Key Features

### Visual Data Browser

- **Collection List**: See all collections at a glance
- **Document Viewer**: View documents in table, card, or JSON format
- **Schema Inference**: Automatically detect document structure
- **Quick Actions**: Edit, delete, duplicate documents

### Search & Filter

- **Full-Text Search**: Search across all fields
- **Field Filters**: Filter by specific fields
- **Advanced Queries**: Build complex queries visually
- **Saved Filters**: Save frequently used filters

### Data Management

- **Bulk Operations**: Select and modify multiple documents
- **Import Wizard**: Upload CSV/XLSX files
- **Export Options**: Download as JSON or CSV
- **Collection Management**: Create and manage collections
