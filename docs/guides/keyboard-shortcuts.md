# Keyboard Shortcuts

Speed up your workflow with keyboard shortcuts. NetPad supports shortcuts for common actions throughout the application.

## Global Shortcuts

These shortcuts work from anywhere in the application.

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + /` | **Open Help Search** (context-aware) |
| `F1` | Open Help Search (alternative) |
| `Ctrl/Cmd + Shift + ?` | Open Help Search (alternative) |
| `Ctrl/Cmd + K` | Quick search / Command palette |
| `Ctrl/Cmd + S` | Save |
| `Ctrl/Cmd + Z` | Undo |
| `Ctrl/Cmd + Shift + Z` | Redo |
| `Ctrl/Cmd + C` | Copy |
| `Ctrl/Cmd + V` | Paste |
| `Ctrl/Cmd + X` | Cut |
| `Ctrl/Cmd + A` | Select All |
| `Escape` | Cancel / Close Modal |
| `Enter` | Confirm / Submit |

:::tip Getting Help
Press `Cmd + /` (Mac) or `Ctrl + /` (Windows/Linux) from anywhere to open the context-aware help search. The help system automatically shows topics relevant to your current page.
:::

## Customizing Shortcuts

NetPad allows customizing some shortcuts:

1. Go to **Settings** → **Keyboard Shortcuts**
2. Find the action you want to customize
3. Click to set new shortcut
4. Press your desired key combination
5. Save changes

### Conflicts
If a shortcut conflicts with an existing one:
- Warning shown
- Choose to replace or cancel
- System shortcuts cannot be overridden

## Platform Differences

### macOS
- Use `Cmd` key
- Option key for alternate functions
- Standard macOS conventions

### Windows/Linux
- Use `Ctrl` key
- Alt key for alternate functions
- Standard Windows conventions

## Tips

### Learn Gradually
- Start with common shortcuts (Save, Undo, Copy/Paste)
- Add more as you get comfortable
- Use the shortcut reference (accessible via `?`)

### Efficiency
- Shortcuts faster than mouse for repetitive tasks
- Combine with mouse for best workflow
- Build muscle memory over time

### Context Awareness
- Shortcuts change based on context
- Check what's available in current view
- Modal shortcuts override base shortcuts

## Help System Shortcuts

Access the in-app help system:

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + /` | Open help search |
| `F1` | Open help search |
| `Ctrl/Cmd + Shift + ?` | Open help search |
| `↑` / `↓` | Navigate help results |
| `Enter` | Select help topic |
| `Escape` | Close help dialog |

The help system is context-aware—it automatically highlights topics relevant to your current page. See [In-App Help](./in-app-help.md) for more details.
