# Keyboard Shortcuts

Speed up your workflow with keyboard shortcuts. NetPad supports shortcuts for common actions throughout the application.

## General Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + S` | Save |
| `Ctrl/Cmd + Z` | Undo |
| `Ctrl/Cmd + Shift + Z` | Redo |
| `Ctrl/Cmd + C` | Copy |
| `Ctrl/Cmd + V` | Paste |
| `Ctrl/Cmd + X` | Cut |
| `Ctrl/Cmd + A` | Select All |
| `Escape` | Cancel / Close Modal |
| `Enter` | Confirm / Submit |

## Form Builder Shortcuts

### Navigation

| Shortcut | Action |
|----------|--------|
| `Tab` | Next field |
| `Shift + Tab` | Previous field |
| `Arrow Up/Down` | Navigate field list |
| `Home` | Go to first field |
| `End` | Go to last field |

| Shortcut | Action |

### View Controls

| Shortcut | Action |

## Customizing Shortcuts

NetPad allows customizing some shortcuts:

1. Go to **Settings** â†’ **Keyboard Shortcuts**
2. Find the action you want to customize
3. Click to set new shortcut
4. Press your desired key combination
5. Save changes

### Conflicts
If a shortcut conflicts with an existing one:
- Warning shown
- Choose to replace or cancel
- System shortcuts cannot be overridden

## Platform Differences

### macOS
- Use `Cmd` key
- Option key for alternate functions
- Standard macOS conventions

### Windows/Linux
- Use `Ctrl` key
- Alt key for alternate functions
- Standard Windows conventions

## Tips

### Learn Gradually
- Start with common shortcuts (Save, Undo, Copy/Paste)
- Add more as you get comfortable
- Use the shortcut reference (accessible via `?`)

### Efficiency
- Shortcuts faster than mouse for repetitive tasks
- Combine with mouse for best workflow
- Build muscle memory over time

### Context Awareness
- Shortcuts change based on context
- Check what's available in current view
- Modal shortcuts override base shortcuts

## Shortcut Reference

Access the full shortcut reference anytime:
- Press `?` or `Ctrl/Cmd + /`
- Opens shortcut overlay
- Shows all available shortcuts for current context

## Accessibility
