Template Gallery

The Template Gallery provides pre-configured form templates to help you get started quickly. Templates are professionally designed starting points that you can customize to match your specific needs.

Overview​

Templates are pre-configured forms with fields, validation rules, and structure already set up for common use cases. Instead of building forms from scratch, you can start with a template and customize it to fit your requirements.

Benefits of Using Templates​

  • Speed: Get started in minutes instead of hours
  • Best Practices: Templates follow industry standards and best practices
  • Common Use Cases: Pre-configured for typical scenarios
  • Learning Tool: See how experienced users structure forms
  • Customizable: Fully editable after applying a template

Accessing the Template Gallery​

The Template Gallery is available when creating a new form:

  1. Navigate to Forms - Click "New Form" from the dashboard
  2. Open Template Gallery - Click the "Templates" tab in the form creation dialog
  3. Browse Templates - Explore templates by category or search
tip

The Template Gallery is also accessible from the Empty Form State when you first open the Form Builder without an existing form.

Template Categories​

Templates are organized into categories to help you find what you need:

Business​

Common business forms for customer engagement and operations:

  • Contact Form - Basic contact information collection
  • Job Application - Comprehensive job application with resume upload
  • Lead Capture - Sales lead information collection
  • Customer Feedback - Gather customer opinions and ratings
  • Quote Request - Product/service quote requests
  • Support Ticket - Customer support request form

Events​

Forms for event management and registration:

  • Event Registration - Event sign-up with ticket selection
  • RSVP Form - Simple RSVP with dietary preferences
  • Volunteer Signup - Volunteer registration with availability
  • Webinar Registration - Webinar sign-up with time zone selection
  • Workshop Registration - Workshop enrollment with prerequisites

Feedback​

Forms for collecting user feedback and surveys:

  • Customer Satisfaction Survey - Comprehensive satisfaction survey
  • NPS Survey - Net Promoter Score survey
  • Product Feedback - Product review and feature requests
  • User Experience Survey - UX research and usability feedback
  • Employee Feedback - Internal feedback collection

Support​

Forms for customer support and service requests:

  • Support Ticket - Detailed support request with priority
  • Appointment Booking - Schedule appointments with availability
  • Service Request - Request services with scheduling
  • Bug Report - Technical issue reporting
  • Feature Request - Product feature suggestions

E-commerce​

Forms for online business operations:

  • Order Form - Product order with payment information
  • Return Request - Product return with reason selection
  • Product Inquiry - Product information requests
  • Shipping Address - Address collection for shipping
  • Warranty Registration - Product warranty registration

Healthcare​

Forms for healthcare and medical use cases (with encryption support):

  • Patient Intake - New patient information collection
  • Health Screening - Health assessment questionnaire
  • Appointment Request - Medical appointment scheduling
  • Consent Form - Medical consent documentation
  • Insurance Information - Insurance details collection
important

Healthcare templates include encryption indicators. These templates are designed to handle sensitive health information and comply with healthcare regulations.

Finance​

Forms for financial services (with encryption support):

  • Expense Report - Employee expense submission
  • Financial Application - Loan or credit applications
  • Invoice Request - Invoice generation requests
  • Payment Information - Payment method collection
  • Tax Information - Tax document collection
important

Finance templates include encryption indicators. These templates are designed to handle sensitive financial information and comply with financial regulations.

Education​

Forms for educational institutions:

  • Course Enrollment - Student course registration
  • Scholarship Application - Scholarship application process
  • Student Information - Student data collection
  • Parent Contact - Parent/guardian information
  • Event Permission - Field trip and event permissions

Real Estate​

Forms for property management and real estate:

  • Property Inquiry - Property information requests
  • Rental Application - Rental application with references
  • Property Viewing Request - Schedule property viewings
  • Maintenance Request - Property maintenance requests
  • Lease Agreement - Lease information collection

Search Forms​

Pre-built search form templates for querying existing data:

  • Customer Search - Search customers by name, email, phone, or other criteria
  • Order Search - Search orders by order number, date, status, or customer
  • Support Ticket Search - Search support tickets by ticket ID, status, priority, or assignee

Search Form Features:

  • Multiple search operators (equals, contains, between, in, regex, etc.)
  • Customizable result display (table, cards, list layouts)
  • Smart dropdowns that auto-populate from database values
  • Pagination and result actions (view, edit, delete, export)
  • Configurable searchable fields per template

Browsing Templates​

Category Filtering​

  1. Select a Category - Click on a category tab to filter templates
  2. View Templates - See all templates in that category
  3. Switch Categories - Click another category to browse different templates

Search Functionality​

  1. Use Search Bar - Type keywords to find templates
  2. Search Results - Templates matching your search appear
  3. Clear Search - Click X or clear search to see all templates

Template Cards​

Each template is displayed as a card showing:

  • Template Name - Descriptive name
  • Icon - Visual identifier
  • Category Badge - Category label
  • Brief Description - What the template is for

Previewing Templates​

Before using a template, you can preview it to see what it includes:

Opening Preview​

  1. Click Template Card - Click on any template card
  2. Preview Dialog Opens - See detailed template information

Preview Information​

The preview shows:

Template Details​

  • Name - Full template name
  • Description - Detailed description of use case
  • Category - Template category
  • Icon - Visual identifier

Template Metadata​

  • Complexity Level - Simple, Moderate, or Advanced
  • Estimated Setup Time - How long to customize (e.g., "2 minutes")
  • Field Count - Number of fields included

Field List​

  • All Fields - Complete list of fields in the template
  • Field Types - What type each field is
  • Field Labels - How fields are labeled
  • Required Indicators - Which fields are required

Encryption Indicators​

For healthcare and finance templates:

  • Encryption Badge - Shows if template includes encryption
  • Compliance Notes - Information about data protection

Preview Actions​

From the preview dialog, you can:

  1. Use Template - Apply template as-is and start customizing
  2. Customize - Apply template and immediately enter customization mode
  3. Close - Return to template gallery

Using Templates​

Method 1: Use Template​

This applies the template and opens the form builder:

  1. Click "Use Template" - In the template preview dialog
  2. Name Your Form - Enter a name for your form
  3. Select Collection - Choose target MongoDB collection
  4. Form Builder Opens - Template fields are loaded
  5. Start Customizing - Modify fields, add/remove as needed

Method 2: Customize​

This applies the template and immediately opens customization:

  1. Click "Customize" - In the template preview dialog
  2. Name Your Form - Enter a name for your form
  3. Select Collection - Choose target MongoDB collection
  4. Customization Mode - Form builder opens with template applied
  5. Edit Immediately - Start modifying fields right away
tip

Both methods result in the same form - the difference is just the initial state when the form builder opens. You can always customize later.

Customizing Templates​

Templates are starting points - they're fully customizable after application.

Modifying Template Fields​

  1. Edit Field Properties - Click any field to edit its properties
  2. Change Field Type - Convert fields to different types if needed
  3. Update Labels - Modify field labels and descriptions
  4. Adjust Validation - Change required status, add validation rules
  5. Reorder Fields - Drag fields to reorder them

Adding Fields​

  1. Open Field Palette - Access field types in the sidebar
  2. Drag New Fields - Add fields not in the template
  3. Configure Fields - Set up new field properties

Removing Fields​

  1. Select Field - Click on a field
  2. Delete Field - Remove fields you don't need
  3. Confirm Removal - Confirm deletion

Best Practices for Customization​

  • Start Simple - Use the template as-is first, then customize
  • Test Changes - Preview form after each major change
  • Keep Structure - Maintain logical field ordering
  • Preserve Validation - Keep important validation rules
  • Document Changes - Note what you changed and why

Template Structure​

Templates follow a consistent structure:

Field Organization​

  • Logical Grouping - Related fields are grouped together
  • Progressive Disclosure - Complex forms may use multi-page structure
  • Clear Labels - Field labels are descriptive and clear
  • Appropriate Types - Fields use the most suitable field types

Validation Rules​

  • Required Fields - Essential fields are marked required
  • Format Validation - Email, phone, and other formats validated
  • Length Limits - Appropriate min/max lengths set
  • Custom Rules - Advanced validation where needed

Best Practices​

Templates demonstrate:

  • User Experience - Good UX patterns and flow
  • Data Structure - Logical MongoDB document structure
  • Accessibility - Accessible form design
  • Mobile Responsive - Works on all device sizes

Template Metadata​

Each template includes metadata to help you choose:

Complexity Levels​

  • Simple - Basic forms with straightforward fields
  • Moderate - Forms with conditional logic or multiple pages
  • Advanced - Complex forms with advanced features

Estimated Setup Time​

  • Quick - 1-2 minutes to customize
  • Standard - 3-5 minutes to customize
  • Extended - 5-10 minutes to customize

Field Count​

  • Small - 5-10 fields
  • Medium - 11-20 fields
  • Large - 21+ fields

Encryption and Compliance​

Healthcare Templates​

Healthcare templates are designed with:

  • Encryption Support - Fields can be encrypted at rest
  • HIPAA Considerations - Designed for healthcare compliance
  • Sensitive Data Handling - Proper handling of PHI (Protected Health Information)

Finance Templates​

Finance templates include:

  • Encryption Support - Fields can be encrypted at rest
  • PCI Considerations - Designed for payment information
  • Financial Data Protection - Secure handling of financial data
important

Encryption must be configured separately. Templates indicate encryption support, but you must enable encryption in your form settings.

Workflow: Creating a Form from Template​

Here's a complete workflow for creating a form from a template:

Step 1: Access Template Gallery​

  1. Navigate to Forms section
  2. Click "Create New Form" button
  3. Click "Templates" tab (or expand "Or start manually" if using AI prompt)

Step 2: Browse and Search​

  1. Browse templates by category (Business, Events, etc.)
  2. Or use search to find specific templates
  3. Review template cards to see options

Step 3: Preview Template​

  1. Click on a template card to preview
  2. Review template details:
    • Name and description
    • Complexity level
    • Estimated setup time
    • Field count
  3. Review field list to see what's included
  4. Check for encryption indicators if needed

Step 4: Apply Template​

  1. Click "Use Template" or "Customize"
  2. Enter form name
  3. Select target MongoDB collection
  4. Click "Create Form"

Step 5: Customize Form​

  1. Review fields - See what the template includes
  2. Edit fields - Modify labels, types, validation
  3. Add fields - Include additional fields as needed
  4. Remove fields - Delete fields you don't need
  5. Reorder fields - Organize fields logically

Step 6: Configure Settings​

  1. Form Settings - Set name, description, thank you message
  2. Access Control - Configure who can submit
  3. Appearance - Customize theme and branding
  4. Advanced - Set up webhooks, encryption, etc.

Step 7: Test and Publish​

  1. Preview Form - See how it looks to users
  2. Test Submission - Submit test data
  3. Verify Data - Check data in MongoDB
  4. Publish - Make form live when ready

Tips and Best Practices​

Choosing the Right Template​

  • Match Your Use Case - Choose template closest to your needs
  • Check Complexity - Start simple if you're new
  • Review Fields - Preview to see if fields match your needs
  • Consider Customization - Factor in how much you'll need to change

Customization Strategy​

  • Start with Template - Use template as foundation
  • Incremental Changes - Make changes gradually
  • Test Frequently - Preview after each change
  • Keep What Works - Don't change things unnecessarily

Template Limitations​

  • Not Perfect Fit - Templates are starting points, not final solutions
  • May Need Changes - Expect to customize for your specific needs
  • Field Types - You may need to change field types
  • Validation - Adjust validation rules as needed
