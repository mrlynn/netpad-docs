## Accessing Templates in NetPad

Templates are available when creating a new form:

1. Navigate to **Forms** section
2. Click **"Create New Form"**
3. Select the **"Templates"** tab
4. Browse or search for templates
5. Click a template to preview, then **"Use Template"**

## Template Categories

Templates are organized into 14 categories:

| Category | Examples |
|----------|----------|
| **Business & Sales** | Contact forms, lead capture, quotes |
| **Healthcare** | Patient intake, medical history |
| **HR & Recruitment** | Job applications, onboarding |
| **Finance** | Expense reports, invoices |
| **Education** | Course enrollment, evaluations |
| **Events** | Registration, RSVPs |
| **Feedback & Surveys** | NPS, CSAT, product feedback |
| **Customer Service** | Help desk, bug reports |
| **E-commerce** | Order forms, returns |
| **Real Estate** | Property inquiries, rentals |
| **Legal** | Client intake, NDAs |
| **Nonprofit** | Donations, volunteer signup |
| **Marketing** | Newsletter signup, webinars |
| **Operations & IT** | IT requests, incident reports |

## Encryption & Compliance

Templates in healthcare, finance, and HR categories include pre-configured encryption indicators for sensitive fields (HIPAA, PCI-DSS, GDPR, SOC2).

See [Encryption Documentation](../security/encryption.md) for setup instructions.
