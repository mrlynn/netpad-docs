# Documentation Images

This directory contains images and screenshots for NetPad documentation.

## Directory Structure

```
docs/
  forms/              # Form-related screenshots
  workflows/          # Workflow screenshots
  data-explorer/      # Data explorer screenshots
  platform/           # Platform screenshots
  api/                # API documentation images
  deployment/          # Deployment screenshots
```

## Adding New Images

1. **Take Screenshot**: Use your preferred screenshot tool
2. **Optimize**: Resize and compress the image
3. **Name Appropriately**: Use descriptive, kebab-case names
4. **Place in Correct Directory**: Organize by feature area
5. **Reference in Docs**: Add image reference in markdown

## Image Guidelines

- **Format**: PNG for screenshots, SVG for diagrams
- **Size**: 1200-1600px width for screenshots
- **Optimization**: Compress before committing
- **Naming**: Use descriptive kebab-case names
